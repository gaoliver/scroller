Infinite Repeater Scroll!

with a random color plus

How it works

This is made by a simple HTML code file with 2 sections. Inside this filme we are going to import the JavaScript file, with the required functions for this effect.

When it reaches end, JavaScript will turn the scroll position to zero, and the change color function will generate random color code to the first section, which is being automatically repeated in the end of page, to make the continuous illusion.

That's it! Just keep scrolling.
